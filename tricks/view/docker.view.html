Pull alpine image

docker pull alpine


Pull web server image

docker pull nginx:alpine


Run docker container and connect to shell

docker container run -i -t --name containerName alpine ash


Run docker container in daemon mode

docker container run -d --name containerName -P ngnix:alpine


Stop existing container

docker stop containerName


Remove existing container

docker rm containerName


List existing running container. It will show linking ports

docker container ls


Link web container to localhost. Replace localhost with the ip of the host that you want to link

elinks --dump http://localhost:port
